# 指令

  • echo : 代表字串格式輸出
  • find : 在指定目錄下尋找檔案和目錄 (-name : 利用名字查詢，-type : 按檔案類型查詢，-size : 按檔案大小查詢)
  • diff : 用於比較檔案的差異，以逐行的方式，比較文字檔的異同處
  • mail : 發送和接收電子郵件
