# Queue 用法

# 定義:

queue (佇列) 是一種線性的、先進先出（First-In First-Out, FIFO）的資料結構 (所以如果要拿出會先拿出陣列最前面的值)。queue 不支援 clear，需要慢慢 pop 掉。
c++ 要使用 queue 容器的話，需要引入的標頭檔：<queue>

# 宣告

# 新增元素 : push

# 刪除元素 : pop

# 元素存取 : front/back

只能操作頭跟尾，不能取得中間的值

# 初始化

因為 queue 沒有 clear，必須慢慢 pop 掉

# deque 用法

# 定義:

deque 是一個雙向佇列 (double-ended queue)，在頭尾兩端插入及刪除十分快速，在中間插入刪除元素比較費時。
c++ 要使用 queue 容器的話，需要引入的標頭檔：<deque>

# 宣告

# 新增元素 : push_front/push_back

# 刪除元素: pop_front/pop_back

# 插入元素 : insert

# 刪除特定位置的元素 : erase

# 初始化 : clear

# 元素存取 : front/back/[i]

# priority_queue

# 定義:

優先佇列（Priority Queue）是一種資料結構，屬於 queue 的一種，但會以優先度區分，優先度越高的越先處理，默認情況下是最大堆，即優先級最高的元素是值最大的元素。常用於需要頻繁插入和刪除元素且希望能夠快速獲取最大或最小值的場景。

# 宣告

# 新增元素 : push

# 取出優先級最高的元素 : top

# 初始化

因為 queue 沒有 clear，必須慢慢 pop 掉
