# Stack (堆疊) 用法

# 定義:

stack 是個很基本也很重要的資料結構，stack 是後進先出原則 (LIFO，Last In First Out)
因為 stack 的後進先出原則，你只能看到最上面 top 的元素是什麼，之前推 push 進去的元素無法去遍歷查看的。

以下 C++ stack 用法與範例將分為這幾部分，

  • 將元素放入 stack 頂端
  • 從 stack 頂端移出元素
  • 取得 stack 裡的元素數量
  • 判斷 stack 是否為空
  • stack 查看最頂端的元素
