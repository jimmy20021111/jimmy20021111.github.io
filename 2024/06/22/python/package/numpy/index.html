NumPy（Numerical Python 的縮寫）是 Python 編程語言的一個核心庫，用於進行科學計算。它的主要功能包括支持大型多維陣列和矩陣運算，以及大量的高效數學函數操作。NumPy 是許多其他科學計算庫（如 SciPy、Pandas、Matplotlib 等）的基礎。

# 填充陣列: pad

填充模式 (mode):

  • constant (預設): 填入 0，可以指定填充數值，也可以指定左右分別數值，constant_values 可以指定要填入什麼數值
  • edge: 左右方分別填入與最左右方相同的數值，也可以稱作為邊緣填充
  • maximum: 填入陣列最大的元素
  • minimum: 填入陣列最小的元素
  • mean: 填入陣列中的平均值
  • median: 填入陣列中的中位數
  • wrap: 填入與原陣列相同的元素，簡單來說就是複製原陣列至左右
  • linear_ramp: 填入逐漸變小的值，直到最小值
  • reflect: 填入與原陣列要保持鏡像反射的方式 (比如：45432 12345 43212)
  • symmetric: 填入與原陣列要保持對稱的方式

# 資料類別 : dtype.kind

  • ?: boolean
  • i : signed integer
  • i1 : int8(-128~127)
  • i2 : int16(-32768～32767)
  • i4 : int32(-2147483648～2147483647)
  • i8 : int64(-9223372036854775808～9223372036854775807)
  • u : unsigned integer
  • u1 : unit8(0~255)
  • u2 : unit16(0～65535)
  • u4 : unit32(0～4294967295)
  • u8 : unit64(0～18446744073709551615)
  • f : floating-point
  • f2 : 半精度
  • f4 : 單精度
  • f8 : 雙精度
  • c : complex floating-point
  • c8 : 雙 32 位複數
  • c16 : 雙 64 位複數
  • b : signed byte
  • B : unsigned byte
  • m : timedelta
  • M : datetime
  • O : object
  • S : String (後面需接數字，代表字串大小，如 S10)
  • U : Unicode
  • V : void
