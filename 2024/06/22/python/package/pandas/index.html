# 空值的類型 : None /np.nan

None 是 Python 本身的一個特殊常量，用於表示缺失值或空值。它在 Python 的任何上下文中都可以使用。
np.nan 是 NumPy 中的一個特殊浮點值，用於表示 "Not a Number"。它主要用於數值計算和數據分析中，表示無效或缺失的數值。

  • 類型
    None 是 NoneType 類型。
    np.nan 是 float 類型。
  • 用途
    None 用於表示一般的空值或缺失值，適用於任何 Python 對象。
    np.nan 用於表示數值計算中的無效值，主要在數值數據和數據分析中使用。
  • 比較
    None 可以通過 is 或 == 來比較。
    np.nan 不能通過 == 來比較，需要使用 np.isnan () 函數。
  • 數據處理
    在 Pandas 和 NumPy 中，np.nan 是處理數值數據時更常用的表示缺失值的方法，因為它與數據框和數組的其他數值類型兼容。
    None 在數值數據處理中不常用，因為它會導致數組的類型變為 object，從而失去數值計算的效率。

# 函式應用在數據中 : apply /agg/transform

  • apply()
    操作對象：行級或列級，對整列或整行應用函數。
    返回結果：取決於 result_type，可以是 DataFrame 或 Series。
    應用場景：通用性強，適用於大多數操作，包括元素級和聚合操作。
  • agg()
    操作對象：行級或列級，對整列或整行應用聚合函數。
    返回結果：聚合結果，通常是數字或簡單的統計結果。
    應用場景：聚合操作，如計算總和、平均值、最大值等。
  • transform()
    操作對象：元素級，對每個元素應用函數。
    返回結果：與原始 DataFrame 或 Series 形狀相同的結果。
    應用場景：元素級操作，如標準化、歸一化、元素級的變換。

# 圖表 : plot

可以創建各類型的圖表，比如線圖、面積圖、散點圖、柱狀圖。

參數 :

  • kind: 指定圖表的類型。包括 line（線圖）、bar（柱狀圖）、barh（橫向柱狀圖）、hist（直方圖）、box（箱形圖）、kde 或 density（核密度估計）、area（面積圖）、pie（餅圖）、scatter（散點圖）等。
  • x,y: x 軸和 y 軸的列名 (df 中的名稱)。
  • xlabel,ylabel : 代表圖中 x 軸、y 軸的名稱。
  • figsize: 指定圖表的大小。
  • use_index: 是否使用 DataFrame 的索引作為 x 軸，默認為 True。
  • title: 圖表的標題。
  • grid: 是否顯示網格線。
  • legend: 是否顯示圖例 (每個圖代表的名稱)。
  • style: 定義線條的風格（僅對線圖有效）。
  • stacked: 對於柱狀圖和面積圖是否堆疊數據。
  • subplots: 如果設為 True，則每列數據將畫在不同的子圖上。
  • layout: 指定子圖的排列方式。
  • sharex, sharey: 是否共享 x 軸或 y 軸。
  • colormap: 用於定義圖表顏色。
  • color: 指定線條或柱狀的顏色。
  • fontsize: 軸標籤的字體大小。
