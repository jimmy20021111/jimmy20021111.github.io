# Set 用法

# 定義:

set (集合) 容器，set 容器裡面的元素是唯一的，具有不重複的特性，而且是有排序的容器，set 容器裡面元素的值是不可修改，但 set 容器可以插入或刪除元素，set 的實作方式通常是用紅黑樹 (red-black tree) 實作的。

# 插入元素 : insert

# 刪除元素 : erase

# 初始化 : clear

# 判斷是否存在

  • count
  • find
  • equal_range

# 插入元素

  • empalce
    與 insert 差不多，但可以確認是否插入成功 (是否重複)
  • emplace_hint
    算是 emplace 的高級版，可以選擇插入的位置

# 查找第一個不小於指定值的元素的位置 : lower_bound

# 查找第一個大於指定值的元素的位置 : upper_bound

# 查看容器能容納最大元素數量 : max_size

實際大小受限於系統内存和其他資源

# 交換兩個容器內容 : swap

# 遍歷容器

由於 set 容器中沒有 at () 成員函數，也沒有 operator []，set 無法單純地隨機讀取某元素，但能透過 iterator 來讀取元素。

# unordered_set 用法

# 定義:

它與 set 的差別只有在於它不會自主排序，是無排序的容器，其用法都一樣。
