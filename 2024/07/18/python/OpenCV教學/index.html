# 色彩模型轉換: cvtcolor

色彩模型有分為四種，cvtcolor 可以自由切換。

  1. RGB (紅、綠、藍): 以 (255,255,255) 表示顏色
  2. RGBA (紅、綠、藍、alpha): alpha 代表透明程度，0 表示完全透明，1 表示完全不透明
  3. 灰階：內容可以包括 256 種灰色
  4. HSV: 以 (100,100,100) 表示顏色

# 色彩模式

  1. cv2.IMREAD_UNCHANGED: 原本圖像
  2. cv2.IMREAD_GRAYSCALE: 灰階圖像
  3. cv2.IMREAD_COLOR: 彩色圖像
  4. cv2.IMREAD_REDUCED_GRAYSCALE_2: 灰階圖像尺寸縮小 1/2
  5. cv2.IMREAD_REDUCED_COLOR_2: 彩色圖像尺寸縮小 1/2
  6. cv2.IMREAD_REDUCED_GRAYSCALE_4: 灰階圖像尺寸縮小 1/4
  7. cv2.IMREAD_REDUCED_COLOR_4: 彩色圖像尺寸縮小 1/4
  8. cv2.IMREAD_REDUCED_GRAYSCALE_8: 灰階圖像尺寸縮小 1/8
  9. cv2.IMREAD_REDUCED_COLOR_8: 彩色圖像尺寸縮小 1/8

# 黑白影像

又稱為閾值二進制，會將圖片分為黑色與白色，分割方式為若像素灰度大於閾值轉換為黑色，反之則為白色。若要將圖片的雜訊降低，可以先將圖片模糊化再轉換，效果會更好。

轉換方式:

  1. cv2.THRESH_BINARY: 大於閾值則等於最大灰度 (可設定)，反之則為 0
  2. cv2.THRESH_BINARY_INV: 與 cv2.THRESH_BINARY 相反
  3. cv2.THRESH_TRUNC: 若大於閾值則等於閾值，反之則不變
  4. cv2.THRESH_TOZERO: 若大於閾值不變，反之則為 0
  5. cv2.THRESH_TOZERO_INV: 若大於閾值等於 0，反之不變

# 自適應二值化

可以進行區域平均值或高斯平均值來提高轉換的效果

轉換方式:

  1. cv2.ADAPTIVE_THRESH_MEAN_C: 區域平均值
  2. cv2.ADAPTIVE_THRESH_GAUSSIAN_C: 高斯平均值

# 偵測滑鼠動作

event 列表 (以數字表示):
0 : cv2.EVENT_MOUSEMOVE (滑動)
1 : cv2.EVENT_LBUTTONDOWN (左鍵點擊)
2 : cv2.EVENT_RBUTTONDOWN (右鍵點擊)
3 : cv2.EVENT_MBUTTONDOWN (中鍵點擊)
4 : cv2.EVENT_LBUTTONUP (左鍵放開)
5 : cv2.EVENT_RBUTTONUP (右鍵放開)
6 : cv2.EVENT_MBUTTONUP (中鍵放開)
7 : cv2.EVENT_LBUTTONDBLCLK (左鍵雙擊)
8 : cv2.EVENT_RBUTTONDBLCLK (右鍵雙擊)
9 : cv2.EVENT_MBUTTONDBLCLK (中鍵雙擊)
10 : cv2.EVENT_MOUSEWHEEL (滾輪上下滾動)
11 : cv2.EVENT_MOUSEHWHEEL (滾輪左右滾動)

flags 列表 (以數字表示):
1 : cv2.EVENT_FLAG_LBUTTON (左鍵拖曳)
2 : cv2.EVENT_FLAG_RBUTTON (右鍵拖曳)
4 : cv2.EVENT_FLAG_MBUTTON (中鍵拖曳)
8~15 : cv2.EVENT_FLAG_CTRLKEY (按 Ctrl 不放)
16~31 : cv2.EVENT_FLAG_SHIFTKEY (按 Shift 不放)
32~39 : cv2.EVENT_FLAG_ALTKEY (按 Alt 不放)

# 偵測鍵盤動作

利用 cv2.waitKey (0) 來偵測，利用鍵盤的 ASCII 碼來轉換，可以用內建的 chr 來查詢。

  1. ESC : 27
  2. 上 : 0
  3. 下 : 1
  4. 右 : 2
  5. 左 : 3

# 獲取影片信息: get

  1. cv2.CAP_PROP_POS_MSEC: 目前影片毫秒數
  2. cv2.CAP_PROP_FRAME_WIDTH: 影片寬度
  3. cv2.CAP_PROP_FRAME_HEIGHT: 影片高度
  4. cv2.CAP_PROP_FPS: 影片 fps
  5. cv2.CAP_PROP_FRAME_COUNT: 影片總幀數
  6. cv2.CAP_PROP_FORMAT: 影片格式
  7. cv2.CAP_PROP_BRIGHTNESS: 攝影機亮度
  8. cv2.CAP_PROP_CONTRAST: 攝影機對比度
  9. cv2.CAP_PROP_SATURATION: 攝影機飽和度
  10. cv2.CAP_PROP_EXPOSURE: 攝影機曝光度
