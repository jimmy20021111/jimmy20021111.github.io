# Line Notify 用法

# 說明:

Line Notify 是一個由 LINE 提供的服務，允許用戶將訊息從應用程式或服務 (GitHub、IFTTT、Python... 等等) 發送到 LINE 群組或個人聊天。這些訊息可以包括文字、圖片、貼圖等，並且可以由第三方應用程式或服務觸發。這種功能對於通知、提醒、警報等應用場景特別有用。

# 建立 Line Notify:

首先要先註冊一個 Line Notify 帳號，並登入才能使用這些服務。
註冊網站: https://notify-bot.line.me/zh_TW/

# 取得 token (權杖)

  1. 在 Line Notify 網站上，在右上角進入「個人頁面」。
  2. 選擇「發行權杖」。
  3. 幫 TOKEN 取名並選擇你想要發送通知的對象（個人或群組），若是群組必須邀請 Line Notify 進群組才能發送訊息。
  4. 複製生成的 Token (必須先儲存起來)，關閉就找不到了！！！

# 發送表情符號:

在發送時可以選擇一些內定的貼圖，會有指定的 stickerPackageId 與 stickerId，只要寫對應的號碼就可以傳送此貼圖。
網址: https://developers.line.biz/en/docs/messaging-api/sticker-list/
