# 真超集合、超集合、真子集合、子集合

  • 真超集合：集合 A 是集合 B 的真超集合，如果 A 包含 B 的所有元素，且 A 中至少有一個元素不在 B 中。換句話說代表 B 是 A 的真子集合。
  • 超集合：集合 A 是集合 B 的超集合，代表 B 可以等於 A 或 B 是 A 的一部分。
  • 真子集合：集合 A 是集合 B 的真子集合，代表 A 的所有元素皆在 B 中，且 B 必須至少有一個元素不在 A 中。
  • 子集合：集合 A 是集合 B 的子集合，代表 A 可以等於 B 或 A 是 B 的一部分。

速記法：有真的 (真超集合、真子集合) 代表不能完全一樣，沒有真 (超集合、子集合) 代表可以完全相同或不同皆可。
